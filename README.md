# Gestion de Presupuesto Personal

_Una pequeña aplicacion donde se puede agregar un presupuesto total, y se mostrara tu presupuesto actual y el restante de lo que te queda,dependiendo si agregas gastos a tu cuenta._
_Tambien cuenta con un cambio de color en alertas cuando el presupuesto restante es menor al 75% y otra cuando es menor al 50%._

## Comenzando 🚀

_Estas instrucciones te permitirán obtener una copia del proyecto en funcionamiento en tu máquina local para propósitos de desarrollo y pruebas._

Facil primero lo clonas con este comando:

## `git clone https://github.com/alejandrobecerraptor/proyecto-presupuesto-semanal.git`

_Despues de eso te toca descargar las dependencias con:_ ⚙️

## `npm install`

_luego tendras que arrancar el proyecto y para eso necesitas ejecutar el comando:_

## `npm start`

y listo disfruta! 🍦
